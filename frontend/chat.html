SSL-Zertifikat installieren

Selbstsigniertes Zertifikat
Muss einmalig installiert werden, damit der Browser die Verbindung als sicher einstuft.
Zertifikat herunterladen (jarvis.cer)
  1. Lade jarvis.cer herunter.
  2. Rechtsklick auf die Datei → Zertifikat installieren.
  3. Wähle Lokaler Computer.
  4. Alle Zertifikate in folgendem Speicher speichern → Durchsuchen → Vertrauenswürdige Stammzertifizierungsstellen.
  5. Fertigstellen + Browser komplett neu starten.
sudo cp jarvis.cer /usr/local/share/ca-certificates/jarvis.crt
sudo update-ca-certificates

Chromium/Chrome benötigt ggf. zusätzlich den Browser-Import (siehe Browser-Tab).

  1. Browser-Einstellungen öffnen.
  2. Nach Zertifikate suchen.
  3. Importieren im Reiter Zertifizierungsstellen.
  4. Datei jarvis.cer auswählen.
  5. Haken bei Dieser CA vertrauen, um Websites zu identifizieren.
